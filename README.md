# aiyouximain-notes

本项目用于归档和发布多个独立的 HTML 页面。这些页面内容互不关联，每个页面均为完整的单文件 HTML，可直接在浏览器中打开使用。

## 项目简介

本仓库主要用作个人或团队在开发、学习过程中产生的零散 HTML 页面的收集与存档。不针对特定网站或服务，仅以文件形式保存和展示。

## 目录结构

```
aiyouximain-notes/
├── index.html          # 仓库主页面（可选）
├── page-001.html       # 示例页面
├── page-002.html
└── ...
```

每个 HTML 文件均为独立页面，文件名以 `page-` 开头并附带数字编号，便于排序和引用。后续新增页面可继续按此规则命名。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖外部资源（字体、图片等除外，但会尽量内联或使用 CDN 引用）。
- 页面功能可能包括：信息展示、工具页面、笔记卡片、学习示例等。
- 每个页面顶部包含简要注释，说明其用途或来源。
- 归档不保证页面长期可用或内容更新，仅作记录用途。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开任意 `.html` 文件即可查看。
3. 也可通过 GitHub Pages 或其他静态托管服务部署后在线访问。

## 维护说明

- 本仓库不定期更新，添加新页面或对已有页面进行修正。
- 不接受外部贡献或 pull request，但欢迎提出 issues 反馈问题。
- 仓库内容不涉及任何商业用途，仅作为个人存档。
